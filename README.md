# MizTagger-pywin

A files tag system based on pywin32.

## How to use

There are two ways to manage the files and tags.

```bash
pip install miz-tagger
```

`MizTagger-register.exe`

`MizTagger-unregister.exe`

or `python /path/to/context_menu.py`

`MizTagger-viewer`
